# Authentication CLI System

A simple command-line authentication system built to practice core programming concepts such as input validation, control flow, and automated testing.

## Features
- Username/password login
- Input validation
- Clear success and error messages
- Automated unit tests

## Files
- auth.py – main authentication logic
- test_auth.py – unit tests for login behavior

## Skills Demonstrated
- Python programming
- Conditional logic
- Defensive programming
- Unit testing (pytest)
- Command-line interaction
